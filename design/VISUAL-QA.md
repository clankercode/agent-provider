# Visual QA ledger

## Capture matrix

- Popup: 360 x 600, light and dark.
- Approval: 760 x 560 desktop and 360 x 640 narrow, light and dark.
- Options: 1440 x 1000 desktop and 390 x 844 mobile, light and dark.
- Operations dashboard: 1440 x 1000 desktop and 390 x 844 mobile, light and dark.

## Delta passes

1. Hierarchy pass: origin/authority prominence, action order, density, and
   responsive structure.
2. Interaction pass: focus rings, hover/disabled/busy/error states, keyboard
   order, and reduced motion.
3. Finish pass: typography, alignment, overflow, contrast, status copy, and
   cross-surface consistency.

Each pass records screenshots or a concrete blocker. Chrome is required;
Firefox is parity-targeted. Safari findings are non-blocking for this release.
