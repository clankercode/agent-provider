# Provenance and relicensing record

## Sitehand source

Max Kaye, the repository owner and authorizing rightsholder, stated on
2026-07-23 that he owns the copyright in the
Sitehand source and explicitly authorized its promotion and relicensing for
Agent Provider under `CC0-1.0 OR Unlicense`. The MIT file in the unpacked
Sitehand bundle was a default prior distribution choice made without consulting
the owner; it is not the selected license for Agent Provider.

Promoted files are inventoried in the repository's `PROMOTED-SOURCES.md`, which
binds the authorization to the source root, containing repository commit,
selected-corpus hash, and intended destination modules, so that code
origin remains auditable. This authorization applies only to owner-authored
Sitehand material. Third-party dependencies, vendored material, generated
artifacts, trademarks, and patent rights are not silently relicensed.

## Release check

Release validation checks the package SPDX expression, root license files,
promoted-source inventory, dependency-license report, and any third-party
notices. If later evidence identifies an independently copyrighted contributor
portion, that portion must retain its terms or be replaced before release.
