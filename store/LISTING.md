# Store listing copy

## Name

Agent Provider

## Short description

Use your own LLM provider with trusted web apps through exact-site consent,
typed tools, quotas, and private audit controls.

## Full description

Agent Provider gives compatible web applications access to a model account you
control without exposing your provider credential to page JavaScript.

You choose the provider, model aliases, limits, trusted sites, and execution
mode. The extension mediates each run with exact-origin grants, bounded
requests, durable per-origin quotas, typed tool proposals, and independent page
and extension approval for sensitive actions.

Key controls include:

- session-only or persistent consent for each exact origin;
- browser-mediated activation of one optional HTTPS site at a time;
- standard, audit-first, and private execution modes;
- metadata-only, bounded audit records with per-site and global deletion;
- OpenAI-compatible, Anthropic-compatible, and Gemini provider profiles; and
- a local operations view for credentials, aliases, quotas, and audit health.

Agent Provider is infrastructure for compatible applications. It does not
replace the application's login, server authorization, validation, or business
limits. Safari is not currently supported.
